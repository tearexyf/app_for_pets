app_for_pets

app_for_pets — это приложение для владельцев домашних животных, созданное на React + TypeScript + Vite.

Приложение помогает хранить информацию о питомцах, отслеживать процедуры, прием лекарств и визиты к ветеринару в одном месте. А также можно записывать важные моменты, прикрепляя к ним фотографии.

Возможности приложения:
- Создание нескольких профилей питомцев;
- Загрузка фотографии питомца;
- Хранение информации:
- имя,
- вид,
- порода,
- дата рождения,
- вес,
- пол,
- заметки,
- Создание процедур и напоминаний;
- Планирование задач по датам и времени;
- Отметка выполненных процедур;
- Статистика выполненных задач;
- Отдельный список процедур для каждого питомца;
- Хранение данных в LocalStorage.

Стек технологий:
- React;
- TypeScript;
- Vite;
- React Router DOM;
- Zustand;
- React Icons;
- CSS.

Запуск проекта:
Установить зависимости:
npm install
Запустить проект:
npm run dev

Цель проекта
Создать удобное мобильное приложение для владельцев домашних животных, которое позволит следить за здоровьем питомцев, не забывать о лекарствах, процедурах и посещениях ветеринара.
Проект разрабатывается как учебный и постепенно развивается в полноценное мобильное приложение.
app_for_pets — это не просто список задач. Это личный дневник здоровья питомца, где каждый профиль хранит историю процедур, лекарств, визитов к ветеринару и помогает владельцу регулярно заботиться о своем животном.
